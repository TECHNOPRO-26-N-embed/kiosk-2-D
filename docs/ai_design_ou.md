# F08 伝票記録 実装関数一覧と役割

---

## 1. get_next_slip_no
- **役割**: 伝票記録CSVファイルから最後の伝票番号を読み取り、次に使うべき伝票番号（連番）を返す。
- **引数**: ファイル名（const char*）
- **戻り値**: 次の伝票番号（int）

## 2. get_current_datetime
- **役割**: システムの現在日時を取得し、指定バッファに"YYYY/MM/DD HH:MM"形式で格納する。
- **引数**: 出力バッファ（char*）、バッファサイズ（int）
- **戻り値**: なし

## 3. create_slip_record
- **役割**: 伝票情報（伝票番号、日時、ユーザーID、商品名、決済方法、レジ番号、返却期限）を構造体に格納する。
- **引数**: 構造体ポインタ、各項目の値
- **戻り値**: なし

## 4. write_slip_to_csv
- **役割**: 伝票情報構造体の内容をCSVファイルに1行追記する。
- **引数**: ファイル名（const char*）、伝票情報構造体ポインタ
- **戻り値**: 正常終了0、異常時-1

---

必要に応じて、入力チェックやエラー処理用の補助関数も追加可能です。

---

# F08 伝票記録 全体フロー

1. 伝票記録処理の開始
2. 伝票記録CSVファイルから次の伝票番号を取得（get_next_slip_no）
3. ユーザーID・商品名・決済方法・レジ番号・返却期限など必要な情報を取得
4. 現在日時を取得（get_current_datetime）
5. 伝票情報を構造体に格納（create_slip_record）
6. 伝票情報をCSVファイルに追記（write_slip_to_csv）
7. 正常終了メッセージまたはエラーメッセージを表示
8. 終了

（main関数やUIから上記の流れで呼び出すことで、伝票記録機能が動作します）
