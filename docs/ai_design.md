[F0（User ID）基本設計]

1. 機能概要
- ユーザーがIDを入力し、該当する個人情報（例：氏名、年齢、連絡先など）を画面に表示する。

2. 入力
- 入力項目：ユーザーID（数値）

3. 処理
  1. ユーザーがIDを入力
  2. システムが user.csv から該当IDのレコードを検索
  3. 一致するレコードがあれば、個人情報を取得
  4. 画面に個人情報を表示

4. 出力
- 個人情報（例：氏名、年齢、連絡先など）

5. 例外処理
- 入力IDが存在しない場合：「該当するユーザーが見つかりません」と表示

6. 画面遷移
  1. ID入力画面
  2. 個人情報表示画面（またはエラーメッセージ）

[F01（ユーザーID入力→個人情報表示）詳細設計アルゴリズム]

1. ユーザーIDを入力する
2. user.csvを開き、全レコードを読み込む
3. 入力されたIDと一致するユーザーを検索する
4. 一致するユーザーがいなければ「該当するユーザーが見つかりません」と表示し、再入力を促す
5. 一致するユーザーがいれば、そのユーザーの個人情報（氏名、年齢、住所、電話番号、性別、貸出状況、備考）を画面に表示する
6. 表示後、次の処理（本の貸出やメニュー選択など）へ進む
